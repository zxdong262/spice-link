# Spice-Link - Remote Desktop Client

Spice-Link is a remote desktop client designed for Spice protocol connections. Connect to your remote servers with ease and manage multiple sessions efficiently.

## Key Features:

- Spice Protocol Support - Connect to any Spice-compatible remote desktop server
- Multi-Tab Interface - Manage multiple connections in a single window with tabbed sessions
- Flexible Grid Layouts - View multiple connections simultaneously with support for 1x2, 2x2, 1x3, 2x3, and more layouts
- Quick Connect - Connect instantly using spice:// URLs with optional password
- Cloud Sync - Optional encrypted sync to keep your settings across devices (personal GitHub Gist)
- Secure - Your privacy matters. No data collection, no tracking - all data stays local unless you enable optional sync

## Perfect for:

- System administrators managing multiple servers
- Remote workers accessing office desktops
- Anyone needing reliable Spice remote connections
